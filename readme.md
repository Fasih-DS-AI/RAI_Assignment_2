## How to Reproduce
1. Install dependencies via `pip install -r requirements.txt`.
2. Download the `jigsaw-unintended-bias-train.csv` dataset from Kaggle and place it in the root directory.
3. Run the notebooks in sequential order (Part 1 through Part 5).
4. Note: Part 5 requires `pipeline.py` to be present in the same directory to execute the moderation routing logic.
